To-Do List Application
A simple and intuitive To-Do List web application built using HTML, CSS, and JavaScript. This app allows users to add, update, and delete tasks efficiently, helping users manage their daily tasks.

Features
Add Tasks: Users can add new tasks to the list by entering the task name.
Delete Tasks: Users can remove completed or unwanted tasks from the list.
Update Tasks: Users can edit existing tasks if changes are needed.
Interactive UI: A clean, responsive interface that allows easy navigation and task management.
Tech Stack
HTML: For structuring the web pages.
CSS: For styling and layout.
JavaScript: For handling user interactions and dynamic content.
